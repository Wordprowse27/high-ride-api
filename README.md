    <p align="center">

## Requirements
- NodeJS( `>= 16.0.0` )

## Installation

```bash
$ npm i
```

## Running the app

```bash
# development
$ npm start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm test
```
